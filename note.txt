### Node.js - URL로 입력된 값 사용하기 :
  URL에 포함된 쿼리 스트링을 해석해서 이용하는 방법

URL 예시)
http://opentutorials.org:3000/main?id=HTML&page=12

1. http : 프로토콜, 통신규칙

2. opentutorials.org : host(domain)
-> 어떤 특정한 인터넷에 연결되어 있는 컴퓨터를 가리키는 주소

3. 3000 : 포트번호
-> 한 대의 컴퓨터 안에 여러대의 서버가 있을 수 있음. 그러면 클라이언트가 접속했을 때 어떤 서버와 통신할지 애매함. 그래서 해당 포트번호를 통해 서버와 통신함.
http는 기본적으로 80번 포트를 사용.(생략가능)

4. main : path
-> 컴퓨터 내의 어떤 디렉토리에 어떤 파일인지 가리킴.

5. ?id=HTML&page=12 : query string
-> 쿼리 스트링의 시작은 '?'로 작성하기로 약속되어있음.
-> 형태 : ?키=벨류&키=벨류...
-> 어떤 데이터를 웹서버로 전달(즉 요청)할 수 있음. 내가 읽고 싶은 정보는 HTML이고 12페이지이다.(예시임)
